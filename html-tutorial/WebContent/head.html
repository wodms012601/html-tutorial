HTML 헤드 요소 <head>...</head>

목록

1. <head> 요소

<head>태그는 브라우즈 정보, 메타 정보, 스크립트, 스타일 시트 등의 요소를 포함할 수 있다.

2. <title> 요소

  • 브라우저 툴바의 제목을 정의한다.
  • 즐겨 찾기에 추가 될 때 페이지 제목을 제공한다.
  • 검색 엔진의 검색 결과에서 페이지의 제목을 표시한다.
  1. <!DOCTYPE html>
  2. <html>
  3. <head>
  4. <title>Title of the document</title>
  5. </head>
  7. <body>
  8. The content of the document......
  9. </body>
  11. </html>

3. <base> 요소

<BASE>태그는 문서안에 있는 상대적인 주소를 분석하는데 어떤 경로를 사용할 것인지를 정의합니다. 문서내에 있는 링크 참조에 경로명을 전부 넣지 않아도 되는 유용한 기능을 한다.
예를들어, 해당 웹페이지를 다른 경로나 웹싸이트에서 사용하지만 해당 웹페이지에서 사용한 이미지는 다른도메인에 속해있다면, 이미지의 경로를 일일이 <img src="http://domain.com/images/imgname.gif">같은 형식으로 사용하지 않고 <img src="imgname.gif">로만 사용해도 가능하게 된다.
  1. <head>
  2. <base href="http://www.w3schools.com/images/" target="_blank">
  3. </head>

4. <link> 요소

<LINK> 태그는 문서와 외부 자원 사이의 관계를 정의한다.
<LINK> 태그는 대부분의 스타일 시트에 연결하는 데 사용됩니다
  1. <head>
  2. <link rel="stylesheet" type="text/css" href="mystyle.css">
  3. </head>

5. <style> 요소

<STYLE> 태그는 HTML 문서에 대한 스타일 정보를 정의하는 데 사용됩니다.
<STYLE> 요소 내에서 HTML 요소가 브라우저에 렌더링하는 방법을 지정합니다
  1. <head>
  2. <style type="text/css">
  3. body {background-color:yellow;}
  4. p {color:blue;}
  5. </style>
  6. </head>

6. <meta> 요소

메타 데이터는 웹 페이지에 표시되지 않지만, 페이지에 대한 메타정보를 정의한다.
메타 정보는 일반적으로 페이지의 설명, 키워드, 문서의 작성자, 마지막으로 수정 및 기타 메타 데이터를 말한다. 메타정보는 브라우즈, 검색엔진 등의 웹서비스에서 사용된다.
<META> 태그는 항상 <head> 요소 내에서 이동합니다.
  1. <META NAME="Generator" CONTENT="페이지 제작툴">
  2. <META NAME="Author" CONTENT="제작자">
  3. <META NAME="Keywords" CONTENT="컴색 키워드">
  4. <META NAME="Description" CONTENT="페이지의 요약설명">
  5. <META NAME="Copyright" CONTENT="저작권 정보">
  6. <META NAME="Subject" CONTENT="홈페이지주제 입력">
  7. <META NAME="Title" CONTENT="홈피이름">
  8. <META NAME="Publisher" CONTENT="만든단체 또는 회사">
  9. <META NAME="Other Agent" CONTENT="홈피 책임자">
  10. <META NAME="Classification" CONTENT="카테고리 분류">
  11. <META NAME="Reply-To(Email)" CONTENT="메일주소">
  12. <META NAME="FileNAME" CONTENT="파일이름 입력">
  13. <META NAME="Author-Date(Date)" CONTENT="제작일">
  14. <META NAME="Location" CONTENT="위치/대한민국인지등 나라이름기록">
  15. <META NAME="Distribution" CONTENT="배포자">
  16. <META NAME="Copyright" CONTENT="저작권">
  18. <META HTTP-EQUIV="content-type" CONTENT="text/html; CHARSET=utf-8"> <!-- /문자인코딩/ -->
  19. <META HTTP-EQUIV="Refresh" CONTENT="60"> <!-- //60초 마다 새로고침 -->
  20. <META HTTP-EQUIV="Refresh" CONTENT="3;url=http://cafe.naver.com/jjdev"> <!-- //주소로 3초후 이동 -->
  21. <META HTTP-EQUIV="Pragma" CONTENT="no-cache"> <!-- //캐쉬가 저장 되지 않게 함 -->
  22. <META HTTP-EQUIV="Expires" CONTENT=" Mon, 08 Sep 2011 12:00:00 GMT"> <!-- //캐쉬 만료일 -->

7. <script> 요소

javascript와 같은 클라이언트 측 스크립트를 정의하는데 사용된다. <script>요소는 이 후 장에서 설명한다.
