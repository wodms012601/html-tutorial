HTML 이미지 <image>

목록

1. image

html문서에서 이미지를 표현하기 위해서는 <img>태그를 사용한다. <img>태그는 닫는 태그가 없이 속성만 존재 한다.

<img>태그 속성
  • src : 이미지의 위치 지정
  • alt : 이미지에 대한 대체 텍스트를 지정
  • width : 이미지의 너비 지정
  • height : 이미지의 높이 지
  1. <img src="smiley.gif" alt="Smiley face" width="42" height="42">

2. 이미지 관련 태그 : <area>태그, <map>태그

이미지의 내부의 일부(부분)를 선택지정하는 태그
  1. <img src="planets.gif" width="145" height="126" alt="Planets">
  2. <usemap="#planetmap">
  4. <map name="planetmap">
  5. <area shape="rect" coords="0,0,82,126" href="sun.htm" alt="Sun">
  6. <area shape="circle" coords="90,58,3" href="mercur.htm" alt="Mercury">
  7. <area shape="circle" coords="124,58,8" href="venus.htm" alt="Venus">
  8. </map>
